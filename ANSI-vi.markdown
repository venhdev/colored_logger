# 🧱 Bảng mã định dạng văn bản ANSI (Terminal)

## 1. Định dạng văn bản

| Mã   | Ý nghĩa                  |
|------|--------------------------|
| 0    | Đặt lại tất cả (mặc định)|
| 1    | **Đậm** / Sáng hơn        |
| 2    | Mờ đi (giảm cường độ)    |
| 3    | _Nghiêng_                |
| 4    | <u>Gạch chân</u>         |
| 5    | Nhấp nháy chậm           |
| 6    | Nhấp nháy nhanh          |
| 7    | Đảo màu (chữ ↔ nền)      |
| 8    | Ẩn văn bản               |
| 9    | ~~Gạch ngang~~           |
| 10   | Phông chữ mặc định       |
| 11–19| Phông thay thế (tuỳ terminal) |
